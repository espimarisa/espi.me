# [espi.me][espi.me]

My personal website and blog, powered by Jekyll.

## Development

1. Ensure you have Ruby and Bundler >=4.0.0 installed and in your path.
2. Run initial setup by running `./init.sh`.
3. To build, run `./init.sh build`.
4. To cleanup previous builds, run `./init.sh clean`.
5. To serve, run `./init.sh serve`.
