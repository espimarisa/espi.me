# [espi.me][espi.me]

My personal website and blog.

## Development

1. Ensure you have Ruby >=4.0 installed.
2. Install Bundler >=4.0 by running `gem install bundler -v "~> 4.0.0"`.
3. Install Gem dependencies by running `bundle install`.
4. Serve locally by running `bundle exec jekyll serve`.
5. Access the development server at `http://127.0.0.1:4000`.
6. To build static output, run `bundle exec jekyll build`.
7. To clean previous builds, run `bundle exec jekyll clean`.
