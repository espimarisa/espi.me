# [espi.me][espi.me]

My personal website and portfolio, built with Astro.

## Usage

If you're trying to host this on GitHub pages, you'll probably want to stop and use something like Vercel.

1. `pnpm i`
2. `pnpm build`
3. `pnpm preview` or `pnpm dev`
